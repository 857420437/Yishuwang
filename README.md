#关于图片的说明

书籍图片大小不统一可能会让浏览器排版发生混乱。图片过大可能会让django的测试服务器无法正常传输数据。
为了获得正确的排版，达到最佳浏览效果，测试时请统一各本书上传的图片大小（标准大小：185X207像素）。图片大小不要超过500K。
关于图片上传后进行规范处理的功能尚待完善。

#关于跨浏览器支持的说明

本网站在各种浏览器近期的版本上通过了测试，但是使用最新的Google Chorme能够达到最佳浏览体验
